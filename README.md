# Typecho-Plugin-HandsomeCall
一款专属于Handsome的信息提示插件
目前版本：`V1.0`
作者：`Wibus`
参考插件：WeiFeng HandsomeNotice

## 食用教程
1. 先下载插件放入/usr/plugins/下，改名为HandsomeCall
2. 在后台管理插件中点按开启插件即可使用
